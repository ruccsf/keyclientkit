# PDF 募集说明书数据提取 —— Demo 验证报告

**日期：** 2026-06-29
**版本：** v1.0

---

## 一、背景

当前系统通过企查查（QCC）MCP API 采集 🟢 公开数据。但在实际使用中发现，QCC 的 `get_financial_data` 接口对资产负债表详细科目的返回**不稳定**——以下 6 个字段时有时无：

- 短期借款
- 长期借款
- 应付债券
- 一年内到期非流动负债
- 应付票据
- 应收票据

代码已内置 🟡 占位回退（`qcc_fetch.py:512-521`），当 QCC 未返回这些字段时，由 Web Search 补充。但 Web Search 路径依赖 AI 代理的提取质量，实际出现过"约 180-200 亿"这类模糊估算文字代替精确数值的问题。

本次 Demo 验证：**是否可以通过下载中国货币网/上交所债券募集说明书 PDF，从中提取详细资产负债表数据来补充 QCC 的缺口**。

### 2.2 PDF 库对比

| 库 | 状态 | 中文支持 | 表格保留 |
|----|:--:|:--:|:--:|
| **pypdfium2** | ✅ 已安装 5.7.1 | ✅ 完美 | ✅ 按行保留 |
| pdfminer.six | ✅ 已安装 20260107 | ✅ 可用 | ❌ 线性化文本，表格结构丢失 |
| pdfplumber | ❌ 缺 cryptography | — | — |
| pypdf | ❌ 未安装 | — | — |

**结论：pypdfium2 是唯一可行的选择。** pdfminer 会将多列表格打散为独立的文本块，导致"短期借款"标签和后方 5 列数值分散在数十行中，无法可靠重建。pypdfium2 按 PDF 原始行输出，每行格式为 `项目名 value1 value2 value3 value4 value5`，可直接解析。

### 2.3 提取技术路线

```
PDF 下载 → pypdfium2.PdfDocument → page.get_textpage().get_text_range()
→ 按行 split → 正则匹配"中文名 + 5 个数字"的行 → 结构化 JSON
```

核心解析代码（~30 行）：
```python
for line in text.split('\n'):
    parts = line.split()
    if len(parts) >= 6:
        vals = parts[-5:]   # 最后 5 列是数值
        name = ' '.join(parts[:-5])  # 前面是科目名
        if all(re.match(r'^\d[\d,.]*$', v) for v in vals):
            result[name] = vals
```

## 三、两种 PDF 对比

### 3.1 信用评级报告（lhratings.com）

| 维度 | 评价 |
|------|------|
| 数据粒度 | ⚠️ 使用聚合指标（短期债务/长期债务/全部债务），不含明细科目 |
| 资产负债表 | ❌ 无完整 BS，只有摘要财务数据表 |
| 可提取性 | ⚠️ 表格用 pdfminer 勉强可提取，pypdfium2 较乱 |
| 适用场景 | 外部评级、关键财务比率、公司概况 |

**不推荐用于补充 QCC 财务缺口。**

### 3.2 债券募集说明书（SSE 上交所）

| 维度 | 评价 |
|------|------|
| 数据粒度 | ✅ 完整的合并资产负债表、利润表、现金流量表 |
| 资产负债表 | ✅ **全部明细科目**（货币资金→资产总计，短期借款→负债合计） |
| 可提取性 | ✅ pypdfium2 完美保留行列结构 |
| 审计状态 | ✅ 经审计的财务数据 |
| 适用场景 | **作为 QCC 财务数据的权威补充源** |

**强烈推荐。** 一份募集说明书 PDF 即可覆盖全部 20+ 个财务指标，且数据是审计过的。

## 五、结论与推荐方案

### 5.1 可行性：✅ 确认可行

- PDF 下载 ✅ | 中文提取 ✅ | 表格解析 ✅ | 科目覆盖率 ✅ | 数据准确性 ✅

### 5.2 推荐数据优先级

```
财务数据： PDF（完整审计） > QCC（不稳定） > Web Search（兜底）
非财务数据： QCC（唯一来源） > Web Search
```

### 5.3 推荐流程

```
Step 1:   QCC 采集（工商/股东/高管/投资/上市/分支/控制人/知识产权/风险/经营）
Step 1.5: PDF 补充（搜索 → 下载最新募集说明书 → pypdfium2 提取 → fill_field 写入）
Step 2:   Web Search（行业分析/供应链/债券/政策等 🟡 字段）
Step 3:   导出报告
```
